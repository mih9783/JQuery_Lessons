20. Фильтр активных элементов формы

Селектор :enabled можно использовать только для выбора HTML-элементов, которые поддерживают атрибут disabled: button, input, optgroup, option, select, и textarea.

Если необходимо выбрать активные элементы формы, то применяется фильтр enabled. Например, нам необходимо выбрать все активные элементы из формы с классом class:
$(".class :enabled");




21. Фильтр неактивных элементов

Противоположный фильтр предыдущему disabled, который выбирает отключенные элементы формы.
$(".class :disabled");
